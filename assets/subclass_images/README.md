Place subclass images in this folder so `app.py` can display them after inference.

Supported formats:
- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

Use the subclass name as the filename, for example:
- `crow.jpg`
- `owl.png`
- `frog.webp`
- `insects.jpg`
- `rain.jpg`
- `sea_waves.png`
- `wind.jpg`
- `crackling_fire.jpg`
- `car_horn.png`
- `engine_idling.jpg`
- `siren.jpg`
- `jackhammer.png`
